Práctica de Arrays

Rompehielo

  1. Crear un archivo index.html, en la misma ubicación / directorio y una carpeta js y dentro de dicha carpeta un archivo app.js.
  2. Vincular el archivo app.js con el archivo index.html.
  3. Dentro del archivo app.js generar:
  1. una variable que contenga un string.
  2. una variable que contenga un array.
  3. una variable que contenga un boolean.
  1. Hacer un console.log() de cada una de las variables anteriormente creadas.
  2. Abrir el archivo con el navegador y verificar que en la consola de las dev tools se imprimen las variables correspondientes.

Tipos de datos

  1. Vamos a declarar algunas variables.
  1. Declara una variable que más tarde contendrá tu nombre.
  2. Declara una variable que más tarde contendrá tu apellido.
  3. Declara una variable que más tarde contendrá tu edad.

Recuerda que al declarar una variable, le indicamos a la PC que debe reservar memoria para almacenar cosas allí!

  1. Vamos a asignar algunos datos.
  1. Dentro de la variable que declaraste para guardar tu nombre, guarda tu nombre como string.
  2. Dentro de la variable que declaraste para guardar tu apellido, guarda tu apellido como string.
  3. Dentro de la variable que declaraste para guardar tu edad, guarda tu edad como número.

Ahora vamos a declarar y asignar en la misma línea.

  1. Declara una variable que se llame nombreCompleto, en ella vamos a guardar las variables nombre y apellido separadas por un espacio.

Ahora intentemos con arrays.

  1. Declara un array con 5 de tus películas favoritas.

Ejercicio complementario

  1. Crear un array que contenga el árbol genealógico de una familia que incluya abuelos, padres, hijos y nietos con la siguiente estructura:
  1. La primer posición tendrá los nombres de los abuelos
  2. La segunda posición a los padres
  3. La tercera a los hijos
  4. La cuarta a los nietos

        Además debe cumplir con los siguientes condiciones:

  1. El array de los abuelos debe ser creado conteniendo a los abuelos.
  2. El array de los padres debe ser creado vació y luego insertar a los padres.
  3. El array de los hijos debe ser creado conteniendo a los hijos.
  4. El array de los nietos debe ser creado vació y luego insertar a los nietos.
  5. Se debe imprimir en consola la cantidad de abuelos, padres, hijos y nietos.
  6. Imprimir en consola el primer abuelo, el primer padre, el primer hijo y el primer nieto.
  7. Imprimir en pantalla si hay un abuelo llamado “Juan”.
