Práctica funciones

¡Para todos los ejercicios, luego de escribir la función, ejecutala con distintos valores!

  1. Crear una función que reciba como parámetros la altura y el ancho de un cuadrado y retorne el área.

  1. Crear una función que reciba como parámetro el radio de un círculo y retorne el área del círculo.

  1. Crear una función que reciba como parámetro una serie de números (al menos 5) y retorne el promedio.

  1. Crea una función que reciba como parámetro un array y retorne la cantidad de elementos que contiene.

  1. Crea una función que reciba como parámetro un array con los días de la semana y retorne la posición que tiene el día “miércoles”.

  1. Crea una función que reciba como parámetro una palabra y devuelva la cantidad de letras que tiene.

  1. Crea una función que reciba como parámetro un precio en pesos argentinos y devuelva el precio en dólares estadounidenses.

  1. Deberás crear una función que reciba un precio como parámetro y retorne el precio mas el IVA (21%).

  1. Escribí  una función con nombre mitad, que tome un número y devuelva su mitad. Tené en cuenta que el operador de división en JavaScript es /.

  1. Hacé una calculadora básica para que calcule automáticamente 2 números que le especifiquemos. Crea 4 funciones que deberán tomar siempre 2 parámetros (numero1 y numero2), estas funciones te sugerimos que se llamen:

  • sumar
  • multiplicar
  • restar
  • dividir

Luego que las hagas ¡probalas!

 

Ejercicio complementario

¿Cómo se usan estas funciones? ¿Cómo hago para pasarle parámetros y obtener un resultado?

¡Simple! Basta con poner el nombre de la función y entre paréntesis el valor correspondiente a sus argumentos:

doble(3);

 

¡Y además podemos usarlas dentro de otras funciones! Por ejemplo:

function doble(numero) {

  return 2 * numero;

}

function siguienteDelDoble(numero) {

   return doble(numero) + 1;

}

O incluso mejor:

function doble(numero) {

  return 2 * numero;

}

function siguiente(numero) {

  return numero + 1;

}

function siguienteDelDoble(numero) {

   return siguiente(doble(numero));

}

Veamos si se entiende; escribí las siguientes funciones:

  • anterior: toma un número y devuelve ese número menos uno
  • triple: devuelve el triple de un número
  • anteriorDelTriple, que combina las dos funciones anteriores: multiplica a un número por 3 y le resta 1
