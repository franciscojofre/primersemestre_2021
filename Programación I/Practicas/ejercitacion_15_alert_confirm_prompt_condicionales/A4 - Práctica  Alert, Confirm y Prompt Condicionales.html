Para la siguiente práctica te pedimos que crees una carpeta nueva.   1. Precalentando

En el primer archivo js diseñemos un programa que le pregunte al usuario por medio de un confirm “¿Sos  culpable?”

En caso afirmativo mostrá un alert con el texto «irás a la cárcel».

En caso contrario mostrá un alert con el texto «irás a casa».

Para verlo funcionar deberás abrir en el navegador el archivo html asociado. Refrescá la página cada vez para que se ejecuten los scripts.

  1. Analizando números

Mediante un prompt pedí al usuario que escriba un número del 1 al 100. Guardá la respuesta en una variable.

Mediante otro prompt pedí al usuario que escriba otro número del 1 al 100. Guardá la respuesta en otra variable.

Creá una variable “resultado” con la suma de los 2 números.

Usando condicionales evaluá:

  1. Si el usuario dejó el campo vacío mostrá un alerta con la leyenda “Ey, necesitamos un número”. ¿Se te ocurre cómo evaluar el campo vacío? Con la ayuda de console.log() fijate qué retorna un prompt al que el usuario no le completó datos.

  1. Si los números ingresados son iguales. mostrá una alerta diciendo “Ey, escribiste el mismo número: elNúmero”.

Una vez más usando condicionales evaluá:

  1. Si el contenido de la variable resultado es menor a 10 mostrá un alerta que diga: “La suma de tus números es suma y es menor a 10”.
  2. Si el contenido de la variable resultado es mayor a 10 y menor a 20 mostrá un alerta que diga “Bien pasaste”.
  3. Si el contenido de la variable resultado es mayor a 20 mostrá un alerta que diga “Te zarpaste”.

  1. Ahora vamos más fuerte. ¿Estás listo?

Dentro del archivo html agregá un H1 que diga: “La magia sucede en la consola!”.

Mediante un prompt solicitá al usuario que ingrese su nombre. Guardá la respuesta del usuario en un variable llamada “nombre”. Imprimí el nombre del usuario en la consola.

Mediante otro prompt solicitá al usuario que ingrese su edad. Guardá la respuesta  del usuario en un variable llamada “edad”. Imprimí la edad del usuario en la consola.

Mediante un confirm preguntale al usuario si le gustan los deportes. Guardá la respuesta del usuario en un variable llamada “fanDeportes”. Imprimí el valor por consola.

Mediante un alert mostrá la lleyenda “Muchas gracias Nombre por responder nuestras preguntas”.

En el mensaje deberá figurar el nombre del usuario. ¿Cómo lo resolverías?

Con todos los datos que tenemos vamos a crear un objeto literal usuario que contenga las propiedades que recolectamos y un método, deportistaProfesional()

Dentro del método debemos evaluar si el usuario es fan de los deportes y es mayor de 18 años. En caso afirmativo la función retornará “Si soy fan de los deportes” de lo contrario retornará “No soy tan fan aún de los deportes”.

Mediante console.log() revisá si tu variable usuario está bien armada.

Probá el método y mostrá el resultado por consola y con un alert.

Repetiremos el pedido 3 veces para conseguir 3 hobbies del usuario. ¿Qué estructura podrías usar para repetir el prompt 3 veces?

Mediante console.log() verificá los datos dentro del array.

Con las respuestas del usuario vamos a construir un array. Para ello necesitaremos que cada una de las respuestas se pushee dentro del array hobbies que creamos antes. ¿Recordás el método de arrays que lo hace?

Agregar los hobbies del usuario al objeto literal ya formado y con console.log()  verificá que tenga ahora el atributo hobbies. ¿Se te ocurre cómo hacerlo usando la notación de punto (.)?
