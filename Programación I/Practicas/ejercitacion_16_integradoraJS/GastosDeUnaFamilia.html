Práctica integradora

Para la siguiente práctica te pedimos que crees una carpeta nueva. Dentro de ella creá un archivo gastos.html y creá una carpeta /js. Dentro de /js colocá un archivo gastos.js.  Luego vinculá cada par de archivos con la etiqueta <script>

Objetivo: generar un reporte de gastos diarios de una familia.

  1. Preguntarle al visitante que recién ingresa si quiere iniciar. Si la respuesta es negativa, deberá mostrarse una alerta que diga "Gracias por haber venido" y luego será direccionado al sitio web de Netflix. Usá el método:

 window.location.href = ‘https://netflix.com’

  1. Si la respuesta es positiva:
  2. Mediante un prompt ingresar la cantidad de integrantes de la familia. Tal vez debas pasar el valor ingresado usando la función Number() para asegurarnos de que se guarde como número. 
  3. Validá que el dato ingresado sea un número y que sea mayor a 2.
    Para chequear que sea número podrías usar la función     typeof(variable) == “number”. 
    Si el usuario coloca algo diferente a un número o es un número menor igual a 2 deberá alertar que el valor necesario es un número mayor a 2 y volver a cargar la página con la función (window.location.reload()).

  1. Con la cantidad de integrantes lista vamos a pedir mediante un prompt para cada uno: nombre y gastos del día validando que:
  1. El nombre no puede estar vacío.
  2. El valor de gasto no puede estar vacío y debe ser un número. Para chequear que sea número podrías usar la función typeof(variable). En cualquiera de esos casos, alertar del error y volver a cargar la página.

  1. Con todos los datos usando un bucle y métodos de arrays creá un array de objetos literales. Ejemplo final:

let integrantes = [

{nombre: "Ada", gastos: 300},

{nombre: "Tim", gastos: 570},

{nombre: "Vincent", gastos: 80},

]

  1. Mostrar dicho array por consola.

  1. Imprimir por consola el nombre y gasto de cada integrante.

  1. Obtener:
  1. Los gastos de toda la familia e imprimirlos por consola.
  2. Si el gasto total supera los $1000 mostrar una alerta con el mensaje “Debemos pedir un préstamo”.
  3. El promedio de gasto del día e imprimirlo por consola.
  4. Si el gasto promedio es inferior a $500 mostrar una alerta con el mensaje “¡Festejamos!”
