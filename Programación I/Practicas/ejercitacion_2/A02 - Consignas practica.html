Ejercitación 2

Haciendo click aquí encontrarás un archivo zip con varias imágenes dentro. Vamos a crear una página con los personajes que allí se encuentran. Te pedimos:

  1. Descargá el zip a tu computadora.
  2. Creá una nueva carpeta para esta práctica.
  3. Dentro de la carpeta creá otra con el nombre “img”
  4. Colocá dentro de la carpeta “img” todas las fotos.
  5. En la carpeta principal del proyecto (la llamamos carpeta raíz) creá un archivo index.html
  6. Agrega un título principal con el contenido “Guía de personajes favoritos”
  7. Luego para cada una de las imágenes que tengas en la carpeta img debés crear la siguiente estructura:
  1. Un título con el nombre del personaje
  2. La imagen.
  3. Un párrafo indicando a qué universo pertenece: marvel, DC. Star Wars.
  4. Un enlace con el texto “ver más” que te lleve a la página principal del universo al que pertenece el personaje. ¿Cómo harías para que el enlace se abra en una nueva ventana? ¿En qué casos sería recomendado usarlo y en cuáles no?
