Práctica integradora

Objetivo

El objetivo de esta práctica será replicar el layout que ves a continuación. Encontrarás aquí los materiales para construirlo.

CONSIGNA

  1. Crear un archivo llamado index.html.
  2. Crear una hoja de estilos llamada style.css la cual deberá estar guardada dentro de una carpeta con nombre css.
  3. Crear otra carpeta llamada images que tendrá las imágenes que usaremos para este ejercicio.
  4. Los contenedores del header ( rgb(239, 76, 39) ) y de todos los productos deben tener un ancho de 1140px.
  5. Cada contenedor de producto debe tener de ancho 380px y 10px de margen. ( Tener en cuenta el modelo de caja →  box-sizing: border-box )
  6. Etiquetas semánticas sugeridas:
  1. <header>
  2. <main>
  3. <nav>
  4. <section>
  5. <article>
  6. <footer>

¡SUERTE!
