Ejercicio integrador

  1. En una carpeta nueva crear un nuevo archivo que llamaremos responsive.html. Crear la carpeta css con un archivo styles.css. Vincular el archivo html con su css.
  1. En nuestro archivo responsive.html Incluir la etiqueta <meta …> para que el viewport del documento se ajuste perfectamente a cualquier resolución de pantalla.
  2. En nuestro archivo css colocaremos los resets globales que venimos usando:

        * { box-sizing: border-box; }

        body { margin: 0; }        

ul, ol { padding: 0; }

li { list-style-type: none; }

a { text-decoration: none; color: #000; }

  1. Dentro del mismo documento responsive.html, vamos a generar:
  1. Un contenedor con un tamaño máximo de ancho de 1200px. Asignarle un borde para poder visualizarlo y centrar horizontalmente en pantalla (margin: 0 auto;).
  2. Dentro del contenedor vamos a generar 3 cajas una al lado de la otra y todas deben ocupar, de alto, la mitad del viewport.
  1. La caja A deberá tener un ancho de 250px expresados porcentualmente y . Aplicar un borde y fondo para visualizarla.
  2. La caja B deberá tener un un ancho de 800px expresados porcentualmente. Aplicar un borde y fondo para visualizarla.
  3. La caja C deberá tener un un ancho de 100px expresados porcentualmente. Aplicar un borde y fondo para visualizarla.
  4. La caja B tendrá un distanciamiento lateral izquierdo y derecho de 25px expresados de manera porcentual.
  5. Todas las cajas tendrán dentro texto de relleno.
  1. Por fuera del contenedor y antes del mismo vamos a crear una cabecera que ocupe de ancho todo el viewport y tenga una altura de 150px.
  1. Aplicar un borde y color de fondo para visualizar la cabecera.
  1. Distanciar la cabecera del contenedor por 15px.
  2. Visualizar desde las dev-tools del browser el documento a través de la opción Modo de diseño adaptable cambiando el ancho y alto de la resolución de ventana.

  1. A continuación aplicaremos al archivo responsive.html los estilos css necesarios (usar media queries) para lograr:
  1. En una resolución igual o inferior a 900px
  1. Que el contenedor tenga 20px de relleno en sus cuatro costados. Y que ocupe el 95% del ancho de la pantalla.
  2. Que las 3 cajas (A, B y C) tengan una altura de 100px
  3. Que la caja A ocupe el 40%  del contenedor y se encuentre a la izquierda.
  4. Que la caja B ocupe el 55% del contenedor y se encuentre a la derecha. Eliminar los márgenes aplicados en el punto 2-B-IV de esta guía.
  5. Que la caja C ocupe el 100% del contenido. Deberá quedar por debajo de la caja A y caja B.
  1. En una resolución igual o inferior a 450px
  1. Que el contenedor ocupe todo el ancho de la pantalla y no tenga relleno.
  2. Que la altura de cada caja dentro del contenedor sea de un quinto de la altura de la pantalla.
  3. Hacer que las 3 cajas ocupen el 100% del ancho de su padre contenedor, ubicándose una debajo de la otra.
